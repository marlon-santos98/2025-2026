How create an aplicattion with Next.js? run the comand "npx create-next-app"
npx -> call the node.js
npx create next app -> Create the aplicattion
This comand must be always initialized when you wanted to create an aplicattion

how test the aplicattion? Run on console the comand: "npm run dev" -> this will start a local web server

How the folders work?
Public -> Store images
src -> main file

export default function Home() -> Sintax used to create and export a function
<></> -> Tag fragment

Layout.js -> Used for reuse a component, like a top or footer

Click Events:
Action onClick: Fired after the button click
'use client': It always must be used when onClick is called
